# 会话打断功能使用说明

## 🎯 功能概述

在 CoPaw Agent 执行任务过程中，用户可以随时打断，Agent 会立即停止当前操作并等待新指令。

---

## 💬 打断指令

### 支持的中英文关键词

| 类型 | 关键词 | 示例 |
|------|--------|------|
| **英文** | `stop` | "stop" |
| **中文** | `停下来` | "停下来" |
| **中文** | `停下` | "停下" |
| **中文** | `停止` | "停止" |
| **口语** | `别做了` | "别做了" |
| **口语** | `别继续` | "别继续" |
| **操作** | `取消` | "取消" |
| **操作** | `中断` | "中断" |

### 智能匹配

- ✅ **精确匹配**: "停下来" → 触发打断
- ✅ **包含匹配**: "先停下来" → 触发打断
- ✅ **短句匹配**: "帮我停下来" → 触发打断
- ❌ **长句不匹配**: "帮我写个文件然后停下来等我通知" → 不触发 (避免误判)

### 打断流程

1. **检测关键词**: `_is_interrupt_keyword()`
2. **取消任务**: `self._reply_task.cancel()`
3. **清理资源**: 等待 0.1 秒清理
4. **返回响应**: "🫡 已停下！超哥，有什么需要调整的？"

## ⚠️ 注意事项

### 1. 打断时机

- ✅ **推荐**: 在 Agent 执行长时间任务时打断
- ⚠️ **注意**: 某些原子操作可能无法立即中断（如文件写入）

### 2. 资源清理

打断后会自动：
- ✅ 取消当前执行的任务
- ✅ 释放锁定的资源
- ✅ 清理临时文件

### 3. 避免误触发

以下情况**不会**触发打断：
- ❌ 长句中包含关键词（>20 字符）
- ❌ 正常指令中的词汇（如"继续工作"）
- ❌ 空消息或 None
