## 📋 快速参考

### 常用命令
```bash
# 启动 CoPaw
copaw app

# 初始化配置
copaw init --defaults

# 下载本地模型
copaw models download Qwen/Qwen3-4B-GGUF

# 查看已下载模型
copaw models

# 清理缓存
copaw clean

# 查看状态
copaw status
```

### 魔法命令（在对话中使用）
- `/reset` - 重置对话上下文
- `/memory` - 查看长期记忆
- `/skills` - 列出已安装技能
- `/config` - 查看当前配置
- `/help` - 显示帮助信息
