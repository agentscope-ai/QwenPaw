# Open Source & Contribution

CoPaw est open source. Le dépôt du projet est hébergé sur GitHub :

**https://github.com/agentscope-ai/CoPaw**

---

## Comment contribuer

Nous accueillons les contributions : rapports de bugs, idées de fonctionnalités, améliorations de la documentation et modifications du code. Vous pouvez ouvrir des issues et des pull requests sur le dépôt ci-dessus. Les directives de contribution, le code de conduite et le processus de participation sont disponibles dans le dépôt — nous attendons votre participation avec impatience !

Nous vous encourageons également à contribuer selon notre [Feuille de route](/docs/roadmap) : les éléments marqués **Recherche de contributeurs** (ex. nouveaux canaux, fournisseurs de modèles, Skills, MCPs ou améliorations de l'affichage/UX) sont de bons points de départ. Consultez la feuille de route pour les priorités actuelles et les plans à long terme, puis rejoignez la conversation sur [GitHub Discussions](https://github.com/agentscope-ai/CoPaw/discussions).
