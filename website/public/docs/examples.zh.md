# 用例与示例

实用的自动化用例和 Prompt 示例，帮助快速上手 CoPaw 的各种功能。

---

## 核心能力

- 📝 文档处理：摘要、翻译、格式转换
- 🔍 信息检索：网络搜索、资料整理
- 💻 编程辅助：代码生成、调试、优化
- 📅 任务管理：日程安排、提醒设置
- 🔄 自动化：定时任务、批量处理

## 自动化用例

### 用例 1：每日新闻摘要

**场景：** 每天早上自动获取行业新闻，生成摘要。

**配置方式：** 使用定时任务（cron）

```json
{
  "type": "agent",
  "name": "每日科技新闻",
  "cron": "0 8 * * *",
  "channel": "console",
  "prompt": "请搜索并总结今天的科技新闻，包括 AI、互联网、芯片领域的重要动态，输出 5 条核心要点。"
}
```

**效果：** 每天早上 8 点自动推送新闻摘要。

### 用例 6：代码辅助开发

**场景：** 编写、调试、优化代码。

**对话示例：**

````
用户：帮我写一个 Python 脚本，定时备份数据库

CoPaw：好的，这是备份脚本：

       ```python
       import os
       import datetime
       import subprocess

       def backup_database():
           timestamp = datetime.datetime.now().strftime("%Y%m%d_%H%M%S")
           backup_file = f"backup_{timestamp}.sql"
           cmd = f"mysqldump -u root -p mydb > {backup_file}"
           subprocess.run(cmd, shell=True)
           print(f"Backup completed: {backup_file}")

       if __name__ == "__main__":
           backup_database()
       ```

       使用方法：
       1. 保存为 backup.py
       2. 添加到 crontab: 0 2 * * * python backup.py
````

## Prompt 示例

以下 Prompt 可直接发送给 CoPaw，根据需要替换 `[括号内容]`。

### 文档处理

**摘要：**

```
请阅读这个文档，帮我总结出 3-5 个核心要点，用简洁的语言描述。
```

**翻译：**

```
请把这段文字翻译成[目标语言]，保持原文的格式和专业术语的准确性。
```

**格式转换：**

```
帮我把这个 [Word/PDF/Markdown] 文件转换成 [目标格式]，保持格式不变。
```

**会议纪要：**

```
请根据以下会议记录，生成结构化的会议纪要，包括：时间、地点、参会人、讨论内容、决议事项、待办事项。
[粘贴会议记录]
```

**报告生成：**

```
请根据以下数据，生成一份 [周报/月报/季报]，包括：工作完成情况、数据分析、问题与风险、下阶段计划。
[粘贴数据或上传文件]
```

### 数据分析

**Excel 分析：**

```
请分析这个 Excel 文件，告诉我：
1. 数据的基本情况（行数、列数、数据类型）
2. 关键指标的统计值（最大、最小、平均、总计）
3. 发现的问题或异常
4. 建议的可视化方式
```

**数据可视化：**

```
请根据这份数据，生成一个 [柱状图/折线图/饼图]，展示 [具体指标] 的分布情况。
```

**数据清洗：**

```
请帮我清洗这个数据：
1. 去除重复行
2. 处理空值（填充或删除）
3. 统一格式
4. 输出清洗后的文件
```

**数据对比：**

```
请对比这两份数据的差异，列出：
1. 新增的数据
2. 删除的数据
3. 变化的数据（包括具体变化）
```

## 使用技巧

### 1. 明确任务目标

❌ "帮我处理一下这个文件"
✅ "帮我把这个 Excel 转成 JSON 格式"

### 2. 提供必要上下文

❌ "写一个脚本"
✅ "写一个 Python 脚本，每天凌晨 2 点备份 MySQL 数据库到 /backup 目录"

### 3. 分步处理复杂任务

```
用户：我要准备下周的项目汇报
CoPaw：好的，我们可以分步完成：
       1. 先整理项目进度数据
       2. 制作 PPT 大纲
       3. 填充详细内容
       4. 生成最终 PPT
       从哪一步开始？
```
