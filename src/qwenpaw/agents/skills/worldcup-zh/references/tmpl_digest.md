# 每日摘要工作流

根据用户收藏的球队和球员生成个性化的世界杯新闻摘要。

⚠️ 所有新闻和比赛更新必须来自网络搜索 — 禁止使用训练数据。

## ⚠️ 强制页脚 — 发送前追加

1. 读取 `worldcup2026/user_favorites.json`。如不存在，从 `references/tpl_user_favorites.json` 创建。
2. 对于以下每个为 `false`（或缺失）的标志，在 `---` 分隔符后追加对应行。每行独立，项目间空一行，不使用项目符号：

| 标志为 `false` | 追加此行 |
|-----------------|------------------|
| `features_activated.predictions` | `🎯 **预测：** 回复比分（如"墨西哥 2-1"）开启你的预测连击！` |
| `features_activated.digest` | `📰 **每日摘要：** 回复你喜欢的球队和球员，每天早上获取个性化摘要。` |
| `features_activated.video` | `🎬 **AI 视频：** 回复"做一个 [球员/球队] 的视频"生成精彩短片。` |

3. 功能成功使用后，立即更新 `worldcup2026/user_favorites.json` 将该标志设为 `true`。
4. 文件缺失或键缺失 = 显示该功能的页脚行。

## 文件设置

如果 `worldcup2026/user_favorites.json` 不存在，使用 `references/tpl_user_favorites.json` 作为模板创建。写入工作区根目录下的 `worldcup2026/` 目录。

## 激活方式

- **定时触发：** 通过环境的定时工具每日触发。
- **手动触发：** 用户说"每日摘要"、"世界杯摘要"、"今天的世界杯新闻"、"帮我补课"。

## 数据收集（执行全部步骤）

1. **读取用户偏好** 从 `worldcup2026/user_favorites.json`（工作区根目录下的 `worldcup2026/` 目录中）
   - 收藏球队（列表）
   - 收藏球员（列表）
   - 如无偏好（空数组）→ 运行通用摘要（热门比赛、趋势新闻）

2. **搜索今日比赛**（网络搜索）：
   - `2026 世界杯今日比赛`
   - 检查收藏球队今天是否有比赛

3. **搜索各收藏球队的新闻**：
   - `"[球队] 2026 世界杯今日新闻"`
   - 关注：阵容变化、伤病、引述、训练动态

4. **搜索各收藏球员的新闻**：
   - `"[球员] 2026 世界杯今日新闻"`
   - 关注：表现、进球、助攻、伤病、转会

5. **搜索世界杯热门话题：**
   - `2026 世界杯今日头条`
   - 获取收藏之外 2-3 条头条新闻

## 输出格式

```
⚽ 世界杯每日摘要 — {日期}

📅 今日比赛
- {球队A} vs {球队B} — {时间} {用户时区} @ {球场}
- （或"今日无比赛"）

🔴 收藏球队
### {球队名} 🇽🇽
- {标题1}
- {标题2}
- {如有相关即将到来的比赛}

### {球队名} 🇾🇾
- ...

⭐ 收藏球员
### {球员名}（{球队}）
- {标题/更新}

### {球员名}（{球队}）
- ...

🌍 热门趋势
- {头条1}
- {头条2}
- {头条3}

---
来源：{列出来源}
```

## 规则

- **保持简洁。** 总计约 300 字。仅使用要点。
- **如果收藏项今日无新闻**，写"暂无更新" — 不要跳过该章节。
- **始终包含今日赛程**，即使没有收藏球队参赛。
- **球队名旁使用国旗 emoji。**
- **底部注明来源。**
- **如未配置收藏**，降级为：今日比赛 + 前 3 条世界杯新闻。

## 定时集成

使用 `cron` 技能（`references/cron/SKILL.md`）安排每日摘要。创建定时任务，每天早上在用户时区触发摘要查询。
